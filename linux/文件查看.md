## 文件内容

 1. cat，取得文件内容。

```html
# cat [-AbEnTv] filename
-n ：打印出行号，连同空白行也会有行号，-b 不会
```

 2. tac， cat 的反向操作，从最后一行开始打印。

 3. more，和 cat 不同的是它可以一页一页查看文件内容，比较适合大文件的查看。

 4. less，和 more 类似，但是多了一个向前翻页的功能。

 5. head，取得文件前几行。

```html
# head [-n number] filename
-n ：后面接数字，代表显示几行的意思
```

 6. tail，是 head 的反向操作，只是取得是后几行。

 7. od，以字符或者十六进制的形式显示二进制文件。
